Academic Projects



Multithreaded Wikipedia ServerDecember 2020

Description:

Server that accepts JSON requests for information on Wikipedia and responds with appropriate information. Designed with a concurrent framework to ensure thread-safety for multiple users.

Technology used:

  • Server and associated programs were written in Java and compiled and tested using Gradle
  • JWiki and Gson API's used to fetch and format appropriate requests
  • Basic grammar was implemented using ANTLR
  • IntelliJ IDE was used to manage and write code
  • Utilized Github to work effectively with other members on seperate branches

Simple RISC MachineDecember 2020

Description:

"Reduced Instruction Set Computer" deployed on the Arm Cyclone V processor. Capable of basic ARM instructions, such as MOV, AND, CMP and more advanced conditional branching processes. Supports communication with connected I/O devices.

Technology used:

  • The RISC was developed using Verilog, compiled on Quartus.
  • Test benches were developed using ModelSim to simulate verilog waveforms
  • The simple RISC machine was hosted on the DE1-SoC FPGA
